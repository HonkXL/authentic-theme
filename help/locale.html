Locale language and format
Learn to manipulate format of date/time across UI, using special tokens.

Localized formats
Because preferred formatting differs based on locale, there are a few tokens that can be used to format a moment based on its locale. There are upper and lower case variations on the same formats. The lowercase version is intended to be the shortened version of its uppercase counterpart.
